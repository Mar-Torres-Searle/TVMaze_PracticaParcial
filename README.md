# TVMaze - Práctica Parcial Web I

Aplicación web para buscar y gestionar series de TV usando la API de TVMaze.

## Funcionalidades

- Búsqueda de series por nombre
- Detalle de cada serie con reparto incluido
- Gestión de favoritos con localStorage
- Filtro de favoritos por nombre y género
- Diseño responsive

## Tecnologías

- React + Vite
- CSS
- TVMaze API

## Instalación

```bash
npm install
npm run dev
```
